# AinaVoice — Waitlist Landing Page

Une **Waitlist Landing Page** pour AinaVoice, la plateforme SaaS de formation en rhetorique et art oratoire propulsee par l'IA.

## Stack technique

- **React** — Composants UI avec gestion du state (`useState`)
- **Vite** — Bundler ultra-rapide
- **Tailwind CSS** — Utilitaires CSS (via `@tailwindcss/vite`)
- **Lucide React** — Icônes SVG

## Design System « Modern Enterprise »

| Élément | Classe Tailwind |
|---|---|
| Fond de page | `bg-zinc-50` |
| Fond carte / header | `bg-white` |
| Texte principal | `text-zinc-900` |
| Texte secondaire | `text-zinc-500` |
| Bordures | `border-zinc-200` |
| Bouton CTA | `bg-zinc-900 text-white` |

Aucune ombre lourde — uniquement des bordures fines pour la profondeur.

## Démarrage rapide

```bash
npm install
npm run dev
```

L'application sera disponible sur [http://localhost:5173](http://localhost:5173).

## Build production

```bash
npm run build
npm run preview
```
